# Providers (V1)

- Supabase — Postgres + RLS (snapshots/diffs/watchers)
- Resend — verification + notification emails
- Pinata — pin snapshot JSON to IPFS (CID stored in DB)
- BaseScan — primary verification/ABI/source metadata
- Sourcify — fallback source verification (fills missing fields only)
- Base RPC — onchain reads and proxy slot checks
- Vercel — deploy + cron polling
