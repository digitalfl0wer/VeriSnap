# Glossary

**Canonical JSON**  
A normalized receipt format that avoids random ordering/format differences.

**Canonical Hash**  
A fingerprint of the canonical JSON. Changes only when the underlying canonical JSON changes.

**Claimed Publisher**  
Publisher identity confirmed by wallet signature.

**Unclaimed Publisher**  
Publisher identity not confirmed (still publishable).

**Diff**  
A computed “what changed” record between snapshots.

**Watch Mode**  
A subscription flow that periodically re-snapshots and alerts when changes occur.

**RPC**  
Remote Procedure Call endpoint used to read blockchain data.
