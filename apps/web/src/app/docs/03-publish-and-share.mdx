# Publish and Share

Publishing stores your receipt snapshot + metadata and creates a shareable page.

## Publish steps

1) Generate a draft
2) (Optional) Claim your publisher
3) Click **Publish**

## What gets published

- the canonical JSON snapshot
- the canonical hash
- project metadata (slug, tx hash/address reference)
- diff history as new snapshots are added over time
